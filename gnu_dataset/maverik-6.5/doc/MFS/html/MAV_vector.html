MAV_vector

Summary

3D vector.

Syntax

typedef struct {
  float x;
  float y;
  float z;
} MAV_vector;

Description

A Maverik vector comprises 3 floats for the (x,y,z) components. MAV_vector's are used to represent both vectors and coordinate positions.
