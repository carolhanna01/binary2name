mav_gfxWindowFontSet

Summary

Define a font.

Syntax

void mav_gfxWindowFontSet(char *s, int font, int *width);

  • char *s
    The name of the font to define

  • int font
    The index to refer to the font in future calls

  • int *width
    The width of each character in pixels
