[ Maverik Level 1 typedefs ]

MAV_box

Summary

Default object class ``box''.

Syntax

typedef struct {
  MAV_vector size;
  MAV_surfaceParams *sp;
  MAV_matrix matrix;
  void *userdef;
} MAV_box;

Description

An axis-aligned box is defined with its center at the origin. It has a dimension, size, along the X,Y and Z axis.
