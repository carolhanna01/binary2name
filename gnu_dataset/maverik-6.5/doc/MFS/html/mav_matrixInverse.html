[ Maverik Level 1 functions ]

mav_matrixInverse

Summary

Invert a matrix.

Syntax

MAV_matrix mav_matrixInverse(MAV_matrix m);

  • MAV_matrix m
    Matrix to invert.

Description

This function computes the inverse of matrix m, and returns the result. If the matrix is singular, the function prints an error message.
