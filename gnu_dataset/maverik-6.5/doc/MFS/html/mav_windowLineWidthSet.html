[ Maverik Level 1 functions ]

mav_windowLineWidthSet

Summary

Set the line width for a window.

Syntax

void mav_windowLineWidthSet(MAV_window *w, float wd);

  • MAV_window *w
    Window.

  • float wd
    Line width.

Description

This function sets the line width for window w to be wd.
