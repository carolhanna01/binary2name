[ Maverik Level 1 functions ]

mav_matrixMult

Summary

Multiply two matrices.

Syntax

MAV_matrix mav_matrixMult(MAV_matrix m1, MAV_matrix m2);

  • MAV_matrix m1
    Matrix to multiply.

  • MAV_matrix m2
    Matrix to multiply.

Description

This function computes m1 *m2, returning the result.
